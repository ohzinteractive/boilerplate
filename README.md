# OHZI Boilerplate

The OHZI Boilerplate is a production-ready template for building high-performance, interactive web experiences using **WebGL**. It serves as a complete starting point for developers who want to create visually stunning and performant web applications with modern 3D graphics capabilities.

This boilerplate is built and maintained by **OHZI Interactive Studio**, combining battle-tested libraries and best practices to jumpstart your next web project.

More info in [docs/intro.md](docs/intro.md)

## Prerequisites

Before you begin, ensure you have met the following requirements:
* You have installed the following version of
    1. Node.js: `18.x.x`
    2. Git: `2.x.x`
    3. Yarn: `1.22.x`

## Installing boilerplate

To install boilerplate, follow these steps:

1. Clone the repository.
2. Navigate to the project directory.
3. Install the dependencies: `yarn install`

## Using boilerplate

To use boilerplate, follow these steps:

1. Start the application: `yarn start`
1. Navigate to: [http://localhost:1234](http://localhost:1234)

## Building boilerplate

To build boilerplate, follow these steps:

1. Build the application: `yarn build`
2. You'll find the built project on `dist` folder.

The resulting build is a standalone web app with static HTML, CSS, Javascript, and Assets files.
