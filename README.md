# OHZI Boilerplate

## Requirements
- Node v14+

## Install
- `yarn install`
- `git submodule add https://github.com/ohzinteractive/core core`
- `git submodule update --init`
- `cd core && git config core.hooksPath git_hooks`

## Usage
- `yarn start`

## Update submodule
- `git submodule update --remote`
